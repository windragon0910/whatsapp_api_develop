# WhatsApp HTTP API
WhatsApp HTTP API that you can configure in a click!

# Environment variables
- `DEBUG=1` - show debug and verbose logs

# Installation
```bash
docker pull ...
```

# Development

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
